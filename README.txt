===========================================
LA PIRÁMIDE DEL OLVIDO - SITIO WEB OFICIAL
===========================================

INSTRUCCIONES DE USO:
---------------------

1. Extrae todos los archivos del ZIP en una carpeta de tu computadora.

2. Abre el archivo "index.html" con tu navegador web favorito:
   - Chrome
   - Firefox
   - Safari
   - Edge

3. El sitio web es completamente funcional sin necesidad de servidor.

ESTRUCTURA DEL PROYECTO:
------------------------

piramide-olvido/
├── index.html          (Página principal)
├── css/
│   └── styles.css      (Estilos del sitio)
├── js/
│   └── script.js       (Interactividad y efectos)
├── images/
│   ├── Logo.png        (Logo del juego)
│   ├── llave-dorada.png
│   ├── llave-plateada.png
│   ├── personaje-principal.png
│   ├── piramide-fondo.png
│   └── textura-piedra.png
└── README.txt          (Este archivo)

CARACTERÍSTICAS:
----------------

✓ Navegación suave entre secciones
✓ Efectos parallax en scroll
✓ Tooltips interactivos al pasar el cursor
✓ Diseño responsive para móviles y tablets
✓ Animaciones y efectos visuales
✓ Formulario de contacto
✓ Menú desplegable con lista de creadores
✓ Botón circular para acceder al juego

NAVEGACIÓN:
-----------

- Inicio: Vuelve al principio de la página
- Juego: Sección con botón para jugar
- Formulario: Sección de contacto
- Creadores: Menú desplegable con los nombres del equipo

PERSONALIZACIÓN:
----------------

Si deseas modificar el sitio:

1. Edita "index.html" para cambiar el contenido
2. Modifica "css/styles.css" para ajustar colores y estilos
3. Actualiza "js/script.js" para cambiar comportamientos
4. Reemplaza las imágenes en la carpeta "images/"
